📊 Marketing Funnel & Conversion Performance Analysis
📌 Project Overview
This project focuses on analyzing marketing funnel data to identify conversion drop-offs, evaluate channel performance, and generate actionable insights to improve lead-to-customer conversion. The analysis is performed using Power BI on a real-world Bank Marketing dataset.

The dashboard provides a clear view of how customers move through different stages of the marketing funnel and highlights opportunities for performance optimization.

🎯 Objectives
Analyze the marketing funnel and conversion flow
Identify key drop-off stages in the funnel
Evaluate channel-wise and campaign-wise performance
Improve lead-to-customer conversion using data-driven insights
🛠 Tools & Technologies
Power BI Desktop – Dashboard design & data visualization
CSV Dataset – Bank Marketing data
Power Query – Data cleaning & transformation
DAX – Calculated measures and KPIs
📂 Dataset
Dataset Source: UCI Machine Learning Repository – Bank Marketing Dataset
https://archive.ics.uci.edu/dataset/222/bank+marketing
Type: Real-world marketing campaign data
Key Fields Used:
Channel
Campaign Contacts
Conversion Status
Account Balance (used as revenue proxy)
Month & Month Number
📈 Dashboard Features
KPI cards for:
Total Leads
Total Conversions
Conversion Rate (%)
Revenue & Revenue per Lead
Funnel and conversion performance analysis
Channel-wise lead and revenue comparison
Campaign intensity vs conversion trend
Interactive slicers (Year, Month, Channel, Campaign)
Clean and professional UI layout
🔍 Key Insights
Certain channels generate higher conversions compared to others
Increasing campaign contact frequency shows diminishing conversion returns
Optimized targeting and channel focus can significantly improve conversion performance
🚀 Outcome
This project helps marketing teams understand customer behavior across the funnel and supports data-driven decision-making to optimize campaign strategies and improve overall conversion efficiency.
